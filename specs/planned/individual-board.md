# Individual Board

Status: Draft
Last updated: 2026-07-16

## Goal

Let a user order saved pedals into one simple composition.

## Functional Requirements

- Horizontal cards, reorder, add, remove, play/pause, loop, and playback position.
- Shared timing and quantized transitions; initial pedal duration is one measure.

## Non-Goals

- Parallel tracks, automation, piano roll, manual notes, individual durations, simultaneous pedal playback.
