# Board Sharing

Status: Draft
Last updated: 2026-07-16

## Goal

Share and safely import an independent board copy through system sharing.

## Functional Requirements

- Versioned custom board package with metadata, order, recipes, effects, and processed images.
- Preview and validate before import; never overwrite existing data automatically.

## Non-Goals

- Mandatory original-image transfer and pre-rendered audio.

## Open Questions

- Package format, security validation, and compatibility policy.
