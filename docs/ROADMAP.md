## Purpose

This roadmap is the living development plan for Snap Battle. Items describe direction, not implementation commitments, unless their status is marked as completed or decided.

## Phase 1: Core PoC

Status: Completed at prototype level.

| Area | Scope | Status |
| --- | --- | --- |
| Foundation Models | Generate structured creature identity from visual metadata | Completed |
| Vision | Classify image labels and infer simple material metadata | Completed |
| Photo Pipeline | Prepare image input, normalize orientation, fingerprint image | Completed |
| Subject Extraction | Extract main subject when supported, fallback when unavailable | Completed |
| Creature Generation | Build `Creature` from observation, draft, validation, and stats | Completed |
| Diagnostics | Track stages, durations, errors, and model availability | Completed |
